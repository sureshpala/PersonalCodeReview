# PersonalCodeReview README

Use this extension to compare branches and then get code reviews from local running LLM (Llamma)

## Features

Connec to git to get list of changes between branches
use that as input to get code reviews

## Requirements

There is a running instance of local LLM (Ollama - llama3) when this extension is executed

## Extension Settings

None
